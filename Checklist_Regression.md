# Checklist de régression – Module connexion

## Avant exécution
- [ ] environnement disponible
- [ ] jeu de données prêt
- [ ] navigateur principal disponible
- [ ] accès clavier vérifié

## Connexion
- [ ] connexion avec identifiants valides
- [ ] erreur si email vide
- [ ] erreur si mot de passe vide
- [ ] erreur si identifiants invalides
- [ ] conservation de l’email après erreur
- [ ] bouton non cliquable en boucle ou non générateur de double action

## Mot de passe oublié
- [ ] lien visible
- [ ] navigation vers l’écran de récupération
- [ ] format email contrôlé
- [ ] message de confirmation compréhensible

## UX / accessibilité de base
- [ ] ordre de tabulation logique
- [ ] focus visible
- [ ] libellés cohérents
- [ ] messages compréhensibles

## Clôture
- [ ] anomalies consignées
- [ ] statut de la campagne mis à jour
- [ ] risques résiduels communiqués
